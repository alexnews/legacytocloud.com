## Part 4: Bug Fixes

### TypeScript Set Iteration Error
**Problem:** Build failed with "Type 'Set<string>' can only be iterated through when using the '--downlevelIteration' flag"
**File:** `frontend/src/app/tips/page.tsx`
**Fix:** Changed spread operator to Array.from()
```typescript
// Before (error)
const categories = [...new Set(tips.map(t => t.category))];

// After (fixed)
const categories = Array.from(new Set(tips.map(t => t.category)));
```
